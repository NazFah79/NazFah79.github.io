What is ALARM?

ALARM stands for A Learning and Responding Matrix. It is a tool that groups NESA's key verbs (directive verbs) by the depth of thinking and response they require — from simple recall all the way to critical evaluation.

💡 The key insight: ALARM is not 5 separate named levels — it's a matrix or spectrum. Key verbs on the left require short, factual answers. Key verbs on the right demand longer, higher-order responses with evidence, reasoning, and judgement. The mark allocation of a question usually signals where on the matrix it sits.

ALARM Level Overview

Identify / Define / List
Recall facts. 1–2 marks.
Describe / Outline
Key features + detail. 2–4 marks.
Explain / Compare
Cause & effect, examples. 4–6 marks.
Analyse / Discuss
Components + implications. 6–8 marks.
Evaluate / Assess
Judgement + criteria. 8+ marks.
Critically Analyse
Challenge assumptions. Extended.

How to use ALARM in an exam

  1. Spot the directive verb in the question (e.g. "Explain", "Evaluate")
  2. Locate it on the ALARM matrix — how deep does your answer need to go?
  3. Check the mark allocation — more marks = more depth needed
  4. Plan your response using VSCC structure (Verb, Subject, Criteria, Content)
  5. Write at the correct depth — don't just describe when asked to evaluate!
⚠️ Common mistake: Students who write "describe" answers for "evaluate" questions miss marks even when their facts are correct. ALARM helps you avoid this.

    ALARM Quick Reference — Mark & Depth Guide

    1–2 marks
    Identify / Define / List / Recall / Name — one or two sentences, factual only
    2–4 marks
    Describe / Outline / Summarise / Classify — key features with some detail
    4–6 marks
    Explain / Compare / Contrast / Apply — cause & effect, reason why, examples required
    6–8 marks
    Analyse / Examine / Discuss / Interpret — components + relationships + implications
    8+ marks
    Evaluate / Assess / Justify / Recommend — judgement with supported criteria, pros/cons
    Extended
    Critically Analyse / Critically Evaluate — all of the above + challenge assumptions, limitations

    Pre-Write VSCC Checklist

    • I have identified the directive verb and located it on the ALARM
    • I know the subject — exactly what the question is about
    • I have noted any criteria/conditions in the question
    • I have planned the content — key points, examples, evidence
    • My response depth matches the mark allocation
    • I have not merely described when a higher-order verb was used
